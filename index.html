Keys4Speed

Este juego consiste en escribir correctamente las letras que aparecen en la pantalla antes de que se acabe el tiempo, que comienza en 30 segundos. El objetivo es lograr la mayor puntuación posible antes de que el tiempo se agote, y la puntuación máxima se guarda al final de cada partida.

Para ello he usado JavaScript para controlar la tecla pulsada por el jugador y dependiendo de si es correcta o no puede el juego reacciona de una manera u otra.

Web de Senderismo

Aplicación de Empresa

Esta aplicación fue creada con una perspectiva de backend donde queríamos controlor el acceso a las distintas partes del sitio web. Según el rol del usuario, podremos modificar los listados (admin, gestor) o sólamente ver los datos (consultor). El registro de un nuevo usuario lo inicia un admin que manda automáticamente un email al nuevo usuario para que se autentifique y complete el registro.

Mayormente está todo hecho con PHP, mostrando las listas con Html y dándole estilo con CSS
